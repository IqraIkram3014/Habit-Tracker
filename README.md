# Habit Tracker

I built a Habit Tracker web app. It is responsive. Uses HTML, CSS and JavaScript.

## Features

- You can add habits to track.

- I also added the option to edit habits.

- Delete habits you no longer need.

- The app has a habit tracking grid.

- It shows your streaks.

- You can navigate through weeks.

- Data is saved locally using LocalStorage.

- The design is responsive.

## Technologies Used

- HTML5

- CSS3

- JavaScript

## How to Run

1. First,. Clone the repository.

You can do this by running:

```bash

git clone https://github.com/IqraIkram3014/Habit-Tracker.git

```

2. Open the project folder in VS Code.

3. Install the Live Server extension.

4. Right click on `index.html`.

5. Click "Open, with Live Server".
